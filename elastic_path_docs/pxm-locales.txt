If your store supports multiple languages, you can localize product names and descriptions. Commerce supports all languages in Unicode 14.0 Character Code Charts. You add and edit localized product names and descriptions using:Commerce ManagerYou can localize product names and descriptions when creating products. By default, there are a number of language codes available. You activate locales in Commerce Manager in SYSTEM > Store Settings > Locales. You can activate a maximum of 5 languages.Create a Product API RequestYou can use create a product API request to add your own custom language codes.
